# meta-analysis/

Reproducible analysis code and results for the systematic review & meta-analysis:

> **"Diagnostic Accuracy of AI Models and cfDNA Methylation for Lung Cancer Screening:
> A Systematic Review and Meta-Analysis"**
>
> Three research questions (RQ1–RQ3) reported in a single paper following PRISMA-DTA 2023.

## Quick start

All scripts are run from the **repository root**.

### 1. Install dependencies

**Python** (≥ 3.9):
```bash
pip install pandas numpy matplotlib
```

**R** (≥ 4.3):
```r
install.packages(c("metafor", "mada", "ggplot2", "dplyr"))
```

### 2. Run in order

```bash
# Step 1 – prepare datasets
python meta-analysis/code/01_data_preparation.py

# Step 2 – RQ1 meta-analysis (R)
Rscript meta-analysis/code/02_meta_analysis_rq1.R

# Step 3 – RQ2 meta-analysis (R)
Rscript meta-analysis/code/03_meta_analysis_rq2.R

# Step 4 – QUADAS-2 figure
python meta-analysis/code/04_quadas2_assessment.py

# Step 5 – PRISMA-DTA flow diagram
python meta-analysis/code/05_prisma_flow.py

# Step 6 – RQ3 narrative table
python meta-analysis/code/06_rq3_narrative_table.py
```

All outputs land in `meta-analysis/data/` and `meta-analysis/results/`.

## Research questions

| RQ | Question | Analysis type | Key output |
|----|----------|---------------|------------|
| **RQ1** | Single-modal imaging AI vs multi-modal fusion AI for lung cancer screening | Random-effects meta-analysis (DerSimonian-Laird) + subgroup analysis | Forest plots, SROC, Deeks funnel |
| **RQ2** | cfDNA methylation alone vs combined with imaging / fragmentomics | Random-effects meta-analysis + meta-regression | Forest plots, bubble plot, SROC |
| **RQ3** | Deep learning + cfDNA methylation: current evidence and future directions | Narrative review (n=6, below quantitative threshold) | Table 3 |

## Key methodological notes

| Item | Detail |
|------|--------|
| Reporting standard | PRISMA-DTA 2023 |
| Statistical model | DerSimonian-Laird random-effects; Knapp-Hartung CI correction |
| Effect measure | Logit-transformed AUC; back-transformed for reporting |
| Variance estimation | `vi = 1 / (n_cases × AUC × (1−AUC))` when SE not reported (`var_approx = TRUE`) |
| Heterogeneity | I², τ², Cochran's Q |
| Subgroups | Validation type (external vs none) · Application setting (screening vs clinical) |
| Publication bias | Deeks funnel + Egger's test (RQ1 imaging-only, n ≥ 10) |
| Quality assessment | QUADAS-2 (4 domains) for core library (n=38); score-mapped for remainder |
| Sparse evidence rule | < 5 studies with AUC → narrative only (applies to RQ3, meth+ctDNA subgroup) |
| PROSPERO | Pre-registration recommended before final analysis |
